# BMI Calculator

A simple Python project that calculates Body Mass Index (BMI) using user weight and height.

## Features
- Calculate BMI instantly
- Shows BMI category

## Technologies Used
- Python

## Run the Project

```bash
python bmi_calculator.py
```

## Example

```bash
Enter your weight in kg: 70
Enter your height in meters: 1.75

Your BMI is: 22.86
Category: Normal weight
```
